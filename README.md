# computer-vision

## Project Details
Building ML models for various computer vision applications from scratch\

Currently working on: creating a convolutional neural network to classify numbers from the mnist dataset

# Project Setup

## Python
Python &ge; 3.10

## Virtual Environment
Create and activate conda environment\
`conda create --name cnn python=3.12`\
`conda activate cnn`

## Launch JupyterLab
from project root:
`jupyter-lab`

## Installing Dependencies 
`pip install -r requirements.txt`
