# computer-vision

## Project Details
For learning about machine learning applications in computer vision

# Project Setup

## Python
Python &ge; 3.10

## Virtual Environment
Create and activate conda environment\
`conda create --name cnn python=3.12`\
`conda activate cnn`

## Launch JupyterLab
from project root:
`jupyter-lab`

## Installing Dependencies 
`pip install -r requirements.txt`
